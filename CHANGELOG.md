# 更新日志 (CHANGELOG)

所有重要的项目变更都记录在此文件中。

## [0.9.2] - 2026-01-09

### 新增
- HTML 导出功能
  - 目录树 + 文件列表双栏界面
  - 搜索支持文件名和文件夹名
  - 表头点击排序（名称/大小/修改时间）
  - 文件夹点击跳转
  - 前进后退导航（鼠标侧键、Alt+方向键）
  - 主题切换（浅色/深色/跟随系统）
  - 底部状态栏显示当前文件夹统计和路径

### 改进
- CSV 导出添加进度弹窗
- 搜索需点击按钮或回车才执行
- 清除按钮智能返回搜索前位置

### 修复
- 修复导出 SQL 查询错误
- 修复点击箭头会切换内容的问题
